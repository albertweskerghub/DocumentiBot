# Informativa sulla Privacy per [ALBERT WESKER]

Questo documento descrive come Albert Wesker raccoglie, utilizza e protegge i dati degli utenti all'interno della piattaforma Discord.

## 1. Dati Raccolti
Il Bot interagisce con la piattaforma Discord tramite le API ufficiali e può raccogliere o elaborare le seguenti informazioni:
- ID Utente di Discord, ID dei Server (Guild ID) e ID dei canali.
- Username e tag Discord.
- Contenuto dei messaggi (esclusivamente al fine di moderazione automatica, filtri anti-spam o calcolo del sistema di livelli/XP).

## 2. Come vengono utilizzati i dati
I dati vengono utilizzati esclusivamente per garantire il corretto funzionamento delle feature del Bot, tra cui:
- Gestione del sistema di livelli e statistiche dei server.
- Esecuzione dei comandi di moderazione (es. log di ban, kick o ammonimenti).
- Invio di messaggi automatizzati (es. messaggi di benvenuto).

I dati NON vengono venduti, condivisi o utilizzati per scopi commerciali o di profilazione pubblicitaria.

## 3. Conservazione dei dati
I dati (come i punteggi dei livelli o le configurazioni del server) vengono conservati in modo sicuro e solo per il tempo strettamente necessario al funzionamento del servizio. 
Gli utenti o gli amministratori dei server possono richiedere la cancellazione dei propri dati in qualsiasi momento contattando lo sviluppatore.

## 4. Contatti
Per qualsiasi domanda o richiesta di cancellazione dei dati, puoi contattare lo sviluppatore tramite Discord all'utente: [@homelander.03].
